# 修仙力量体系

## 修炼境界（由低到高）

| 境界 | 寿命 | 特征 | 突破关键 |
|------|------|------|----------|
| 练气期（1-13层） | 约150年 | 引气入体，感应天地灵气 | 灵根资质、功法、灵药 |
| 筑基期（初/中/后/大圆满） | 约300年 | 凝练真元，可御剑飞行 | 筑基丹、心境 |
| 结丹期（初/中/后期） | 约500年 | 丹田结金丹，神识大增 | 金丹天劫、悟道 |
| 元婴期（初/中/后期） | 约1000年 | 元婴离体，可夺舍重生 | 元婴天劫、道心 |
| 化神期（初/中/后期） | 约2000年 | 领悟天地法则，可撕裂空间 | 化神天劫、法则感悟 |
| 炼虚期 | 约3000-4000年 | 虚空挪移，威压覆盖百里 | 炼虚天劫 |
| 合体期 | 约5000年以上 | 天人合一，法则具象化 | 合体天劫 |
| 大乘期 | 近万年 | 接近仙人之力 | 大乘天劫 |
| 渡劫期 | — | 渡过天劫即可飞升仙界 | 最终天劫 |

## 修炼要素

### 功法
- 分品级：人阶、地阶、天阶、仙阶
- 属性对应五行：金木水火土，及特殊属性（雷、风、冰、暗等）

### 灵药/丹药
- 按境界分级，高阶丹药炼制极难
- 关键丹药：筑基丹、结丹丹、元婴丹等

### 法宝
- 分级：法器 → 灵器 → 古宝 → 灵宝 → 仙器
- 需滴血认主、温养祭炼

### 神通/术法
- 基础术法：火球术、御风术、隐身术等
- 高阶神通：通常需特殊传承或血脉

### 灵兽/灵虫
- 可契约驱使
- 部分灵虫（如噬金虫）极其珍贵

## 战斗体系
- 低阶：法术对轰，符箓辅助
- 中阶：法宝为主，神通为辅，阵法困敌
- 高阶：法则之力碰撞，领域对抗，本命法宝

## 写作注意
- 境界碾压有限度：差一大境界基本无法抗衡
- 但主角可通过法宝、灵虫、阵法等越级战斗
- 突破场景需要仪式感：天地异象、灵气汇聚、心魔考验
