# 文风指南 — 玄幻修仙网文写作规范

## 核心文风
- **张弛有度**：紧张场面用短句，铺垫场面可适当延展
- **以小见大**：通过细节展现宏大世界观
- **沉浸感优先**：让读者忘记在看书，像亲眼目睹

## 绝对禁止的AI八股词
以下词汇/句式在正文中**严禁使用**：
- "总之"、"综上所述"、"总而言之"
- "值得一提的是"、"不可否认"
- "某种程度上"、"从某种意义上说"
- "毋庸置疑"、"显而易见"
- "与此同时"（用"这时"或直接转场替代）
- "事实上"、"实际上"（直接陈述事实）
- "不禁"（少用，改为具体动作描写）
- 过度工整的排比句（三段以上的排比）
- 章末总结段（"本章讲述了..."）

## 对话写作规范
- 每个角色的说话方式必须有辨识度
- 对话不要总是"某某说道"，用动作、表情替代说话标签
- 避免长篇大论的对话独白
- 高手说话简练有威压，低阶弟子说话可稍活泼

## 战斗场面
- 有视觉冲击：法术特效、灵光流转
- 有策略感：不要无脑对轰
- 有危机感：不要总是轻松碾压
- 短句叠加制造紧迫感
- 大招释放时单独成段

## 修炼/突破场面
- 必须有仪式感
- 天地异象不能少
- 内心体悟要有哲理但不说教
- 突破成功后的感知变化要具体

## 爽点设计模板
1. **打脸流**：被人轻视 → 展露实力 → 众人震惊 → 对手后悔
2. **奇遇流**：陷入绝境 → 意外发现 → 获得宝物/传承 → 实力飞跃
3. **升级流**：苦修积累 → 瓶颈挣扎 → 顿悟/机缘 → 突破成功
4. **复仇流**：被欺压/追杀 → 隐忍修炼 → 归来报仇 → 快意恩仇

## 章节结构
- **标准字数**：2000-3000字/章
- **开头**：前3句必须抓住注意力（动作/对话/悬念）
- **中间**：至少一个情绪起伏
- **结尾**：必须有钩子（悬念/转折/危机）

## 段落节奏
- 紧张时：短段落、短句子、甚至单字成段
- 铺垫时：中等段落，可以有环境描写
- 高潮时：交替使用短长句，制造节奏感
- 严禁出现超过200字的纯描写段落（会被读者跳过）
